## Quick Start

### USB Connection
1. Enable **Developer Mode** and **USB Debugging**  
2. Connect your device via USB  
3. Launch Escrcpy and start mirroring  

### Wireless Connection
1. Enable **Wireless Debugging** on your device  
2. Use QR code scanning or enter device IP  
3. Start mirroring
